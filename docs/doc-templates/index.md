# Templates

## Live walkthrough template
- Status, target, scope, reset time
- Recon baseline (readable bullets)
- Current plan
- Progress timeline (UTC)
- Current assessment
- Next actions

## Walkthrough quality standard
- Include at least 2-3 snapshots per walkthrough.
- Use snapshots only for meaningful milestones:
  - service discovery / recon proof
  - exploitation result (or blocker)
  - privilege escalation or final proof
- Keep text human-readable (short sections, bullets, spacing).

## Methodology checklist
- Reproducible commands
- Decision rationale
- Only useful findings and outcomes
